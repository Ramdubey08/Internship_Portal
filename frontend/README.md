# Internship Portal Frontend

React frontend for the Internship Portal application.

## Setup

```bash
npm install
npm start
```

## Available Scripts

- `npm start` - Run development server on http://localhost:3000
- `npm build` - Build for production
- `npm test` - Run tests

## Environment Variables

Create `.env` file:
```
REACT_APP_API_URL=http://localhost:8000/api
```

## Features

- User authentication (login/register)
- Browse and search internships
- Student applications
- Company internship management
- Profile management
- JWT token handling with auto-refresh
